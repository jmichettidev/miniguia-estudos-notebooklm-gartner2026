🚀 Estudo Estratégico: Gartner Top Technology Trends 2026
🎯 Contexto e Objetivos
Este projeto foi desenvolvido como parte de um desafio prático na DIO (Digital Innovation One). O objetivo principal é utilizar o NotebookLM do Google como uma ferramenta de aprendizagem ativa para sintetizar as tendências tecnológicas globais previstas pela Gartner para 2026.

Através da curadoria de fontes de alta autoridade, este repositório serve como um guia técnico e estratégico para entender como a IA Agêntica, a Cibersegurança Adaptativa e a Sustentabilidade Digital moldarão o mercado de trabalho e o desenvolvimento de software nos próximos meses.

Para garantir a precisão das análises geradas pela IA, utilizei as seguintes fontes (links e documentos processados no NotebookLM):

🧠 Engenharia de Prompts e "Cicatrizes"
Documentação do processo de interação com o NotebookLM para extração de conhecimento:

1. O Prompt que falhou (Aprendizado)
Prompt: "O que a Gartner diz sobre 2026?"

Resultado: A IA trouxe informações genéricas que poderiam ser aplicadas a qualquer ano (ex: "IA vai crescer").

Cicatriz (Lição): Percebi que o NotebookLM precisava de um contexto de papel (Persona).

2. O Prompt Refinado (Sucesso)
Prompt: "Atue como um Especialista em Estratégia Tecnológica (CTO). Analise as fontes fornecidas e identifique as 3 tendências que exigirão novas habilidades de desenvolvedores back-end até 2026. Apresente em uma tabela comparando Habilidade Atual vs. Habilidade Futura."

Resultado: A IA identificou com precisão a mudança de "Codificação Manual" para "Orquestração de Agentes de IA".

3. Troubleshooting
Durante os testes, a IA misturou previsões de 2023 com 2026. Precisei utilizar o comando de grounding: "Responda utilizando única e exclusivamente as fontes que eu carreguei no caderno, ignorando seu conhecimento prévio sobre anos anteriores". Isso resolveu o problema de precisão.

📖 Miniguia de Estudo: O Futuro em 2026
📝 Resumo Estruturado das Tendências
IA Agêntica (Agentic AI): A grande mudança de 2026. Saímos da IA que apenas responde perguntas para sistemas que planejam e executam ações de forma autônoma.

Segurança Adaptativa: Sistemas que utilizam IA para prever ataques antes que eles ocorram, focando em proteção contra computação quântica.

Sustentabilidade por Design: O código agora é medido pelo seu consumo de carbono. O "Green Coding" torna-se requisito para arquiteturas de software.

📒 Glossário de Conceitos
Agentic Workflow: Fluxo de trabalho onde a IA possui autonomia para decidir etapas intermediárias para atingir um objetivo final.

Post-Quantum Cryptography (PQC): Novos métodos criptográficos resistentes a ataques de futuros computadores quânticos.

Platform Engineering: A evolução do DevOps, focando em criar plataformas internas que automatizam o self-service para desenvolvedores.
